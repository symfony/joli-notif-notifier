JoliNotif Notifier Bridge
=========================

Provides a [JoliNotif](https://github.com/jolicode/JoliNotif) integration for
the Symfony Notifier Component.

DSN example
-----------

```
JOLINOTIF_DSN=jolinotif://default
```
